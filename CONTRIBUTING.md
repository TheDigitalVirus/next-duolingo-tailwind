# Contribuindo para o Next Duolingo Tailwind Clone

Primeiramente, obrigado por dedicar seu tempo para contribuir! ❤️

Este é um projeto **educacional** focado na prática avançada de Tailwind CSS, React e Next.js. Se você gosta do projeto mas não tem tempo para contribuir, existem outras formas simples de apoiá-lo:

- Dê uma estrela (star) no projeto no GitHub ✨
- Compartilhe o projeto em suas redes sociais
- Mencione o projeto em meetups locais ou para amigos/colegas

## Encontrou um bug?

1. **Verifique se o bug já foi reportado** acessando a seção [Issues](https://github.com/TheDigitalVirus/next-duolingo-tailwind/issues) do repositório.
2. **Se não encontrar um issue existente**, [abra um novo](https://github.com/TheDigitalVirus/next-duolingo-tailwind/issues/new).
3. **Inclua informações claras**:
   - Título descritivo
   - Passos para reproduzir o problema
   - Comportamento esperado vs. comportamento atual
   - Capturas de tela, se aplicável
   - Ambiente (navegador, sistema operacional, versão do Node.js)

## Desenvolveu uma correção para um bug?

1. **Abra um Pull Request (PR)** no GitHub com a correção.
2. **Descreva claramente** no PR:
   - O problema que está sendo resolvido
   - A solução implementada
   - Como testar a correção
3. **Referencie o número do issue** relacionado, se aplicável.

## Quer adicionar uma nova funcionalidade ou alterar uma existente?

1. **Sugira sua ideia** iniciando uma discussão ou abrindo um issue antes de começar a codificar.
2. **Considere o propósito educacional** do projeto: o foco é demonstrar técnicas avançadas de Tailwind CSS, componentização React e boas práticas de frontend.
3. **Forking é encorajado!** Este projeto serve como material de aprendizado - sinta-se à vontade para criar sua própria versão.

## Tem dúvidas sobre o projeto?

- Para **dúvidas técnicas** sobre configuração ou funcionamento, [abra uma discussão no GitHub](https://github.com/TheDigitalVirus/next-duolingo-tailwind/discussions).
- Consulte o [README.md](https://github.com/TheDigitalVirus/next-duolingo-tailwind/blob/main/README.md) para instruções detalhadas de instalação e execução.
